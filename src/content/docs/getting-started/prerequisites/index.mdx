# Before you begin

## What you should have
- An active NarraMap account.
- A project with at least one journey.
- Permission to edit stories and documentation.

## Recommended setup
- Connect your git repository if you want commit history in the project.
- Decide whether your team will use sprint branches, release branches, or both.
- Set up SmokeHub targets before your first release.
