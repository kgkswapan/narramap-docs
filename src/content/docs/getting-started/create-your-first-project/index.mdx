# Create your first project

## Steps
1. Open the projects area and create a new project.
2. Add a short summary that explains the product goal.
3. Define one or two journeys so the story board has structure.
4. Add your first stories and assign rough priorities.

## Tip
Keep the first setup lightweight. NarraMap becomes more valuable as your team adds real stories, notes, and release context.
