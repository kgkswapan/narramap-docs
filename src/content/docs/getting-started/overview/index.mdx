# Welcome to NarraMap

NarraMap helps a product team plan work, keep documentation current, and ship with cleaner release discipline.

## What you can do here
- Organize work into stories, sprints, and versions.
- Keep living user documentation next to the product.
- Review and publish docs without losing draft history.

## Good first steps
1. Create a project and confirm the project goal.
2. Add journeys and a few starter stories.
3. Open DocHub and shape your first user guide.
